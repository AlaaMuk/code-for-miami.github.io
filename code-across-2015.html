Code Across 2015

Saturday, February 21st @ 10:00AM!

CodeAcross is a weekend of civic hacking events hosted by nodes of the Code for America network around the world. CodeAcross is timed to coincide with both the last weekend of the Code for America Fellows residency and International Open Data Day. The theme for CodeAcross 2015 is Principles for 21st Century Government. Who should get involved? Urbanists, civic hackers, government staff, developers, designers, community organizers, and anyone with the passion to make their city better.
